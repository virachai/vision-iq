# Development Workflow

Follow these steps to build and test the application.

1. Install dependencies (if needed)

```bash
pnpm install
```

2. Run development build
   // turbo

```bash
pnpm run build
```

3. Run unit tests
   // turbo

```bash
pnpm run test
```

4. Run linting
   // turbo

```bash
pnpm run lint
```
